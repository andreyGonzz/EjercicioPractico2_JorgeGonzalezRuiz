Crear Nuevo Rol

Roles del Sistema: Los roles determinan los permisos y accesos de los usuarios.
Roles actuales: ADMIN, PROFESOR, ESTUDIANTE
El nombre debe ser único y descriptivo. Se recomienda usar MAYÚSCULAS.
Proporciona una descripción clara de las funciones y permisos de este rol.
Ejemplos de Roles
COORDINADOR
Gestiona aspectos académicos específicos, supervisa profesores y estudiantes en áreas determinadas.
ASISTENTE
Brinda soporte administrativo, acceso limitado a funciones específicas del sistema.
INVITADO
Solo lectura, acceso muy limitado para visualizar información específica.
MODERADOR
Gestiona contenido y supervisa interacciones, permisos intermedios.
Importante:
  • Una vez creado, el nombre del rol puede modificarse pero afectará a todos los usuarios asignados
  • Los roles con usuarios asignados no pueden eliminarse
  • Los permisos específicos se configuran en el código de la aplicación
